Battle City - Python
Классическая аркадная игра, портированная с JavaScript на Python. Управляйте танком, уничтожайте врагов и избегайте их атак.

Управление
WASD - движение
Пробел - стрельба
R - перезапуск

Особенности
Разные типы стен: кирпичные (разрушаемые), стальные (прочные), лес (проходимый), вода (непроходимая)
Система очков и жизней
ИИ для вражеских танков
Столкновения и физика
